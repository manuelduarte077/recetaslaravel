# Sistema Web de Recetas

Sistema Web de Recetas de comidas y bebidas

## Installation

Instalar los paquetes de composer

```bash
composer install
```

Despues de instalar los paquetes


```bash
composer update 
```

Despues inatalamos los paquetes de node

```bash
 
 npm install

 npm run dev

```

## para arrancar el proyecto localmente

```laravel

1. php artisan migrate:fresh --seed

2. php artisan serve

```
